CMPTokenRequest

It is responsible for the authentication process.

– authentication:onFailure:

Performs authentication request, by passing the encryption-key, used to encrypt payment data, as parameter.

- (void)authentication:(void ( ^ ) ( CMPAuthentication *authentication ))onSuccess onFailure:(void ( ^ ) ( NSError *error ))onFailure

Parameters

onSuccess

Success completion block. It includes the CMPAuthentication object as parameter, which contains all the auth info that will be injected inside the payment request.
onFailure

Failure completion block. It includes the error as parameter.
